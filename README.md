# Understand-Vectors
Dokumentasi proses belajar vektor dari dasar secara bertahap sebagai upaya memahami struktur arah, ruang, dan representasi matematisnya. Repo ini bukan sekadar kumpulan rumus tentang komponen dan operasi, melainkan perjalanan memahami bagaimana vektor bekerja secara aljabar sekaligus geometris. Repo ini berisi catatan pribadi, pembuktian ulang, eksplorasi konsep, serta berbagai kebingungan dan insight yang muncul selama proses memahami vektor secara aljabar dan geometris.
