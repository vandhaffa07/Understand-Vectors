# Understand-Vectors
Dokumentasi proses belajar vektor dari dasar secara bertahap sebagai upaya memahami struktur arah, ruang, dan representasi matematisnya. Repo ini berisi catatan pribadi, pembuktian ulang, eksplorasi konsep, serta berbagai kebingungan dan insight yang muncul selama proses memahami vektor secara aljabar dan geometris.
